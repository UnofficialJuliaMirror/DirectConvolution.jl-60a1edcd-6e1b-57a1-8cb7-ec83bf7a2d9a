DirectConvolution.direct_conv — Method.
direct_conv(...)

Compute convolution.

Here's some inline maths: $\sqrt[n]{1 + x + x^2 + \ldots}$.

Here's an equation:

\[\frac{n!}{k!(n - k)!} = \binom{n}{k}\]

This is the binomial coefficient.

Return γ, a created vector of length identical to β one.
